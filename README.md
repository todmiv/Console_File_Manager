# Console_File_Manager

Консольный файловый менеджер на Python. Позволяет управлять файлами и папками через командную строку.

## Функции

- Создание папок
- Удаление файлов и папок
- Копирование файлов и папок
- Просмотр содержимого директории
- Сохранение содержимого директории в файл
- Просмотр только папок или только файлов
- Информация об операционной системе
- Информация о создателе
- Викторина по датам рождения известных людей
- Симулятор банковского счета
- Смена рабочей директории

## Установка

1. Клонируйте репозиторий:
   ```
   git clone https://github.com/todmiv/Console_File_Manager.git
   cd Console_File_Manager
   ```

2. Установите зависимости:
   ```
   pip install -r requirements.txt
   ```

## Запуск

```
python main.py
```

Следуйте меню для выбора действий.

## Структура проекта

- `main.py` - главный файл с меню
- `create_folder.py` - создание папки
- `delete_file_or_folder.py` - удаление файла/папки
- `copy_file_or_folder.py` - копирование файла/папки
- `view_and_save_directory_contents.py` - просмотр и сохранение содержимого
- `view_only_folders_or_files.py` - просмотр папок или файлов
- `seven_functions.py` - дополнительные функции (ОС, викторина, счет, директория)
